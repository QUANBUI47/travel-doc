# [Client/App] Kho tính năng: Công cụ Tìm kiếm (Search Engine UX)

Bộ công cụ tìm kiếm và lọc Tour dành cho khách hàng tìm kiếm tour nhanh nhất.

## 1. Tìm kiếm Đa năng (Omni-Search Window)
- **Tính năng**: 
    - Bàn phím gõ gợi ý (Fuzzy Search).
    - Hiển thị ảnh nhỏ (Thumbnail) trong danh sách kết quả gợi ý.
- **Phản hồi**: Tốc độ phản hồi < 200ms bằng cách sử dụng debounce gõ.

## 2. Bộ lọc Thông minh (Smart Filters)
- **Tính năng**: 
    - Lọc theo Vùng miền.
    - Lọc theo Khoảng giá (Price Slider).
    - Lọc theo Đánh giá của khách hàng (Rating).

## 3. Bản đồ Tương tác (Interactive Map View)
- **Tính năng**: 
    - Hiển thị Pin tour/địa danh trên bản đồ.
    - Khi di chuyển bản đồ, hệ thống tự động fetch và hiển thị Tour tương ứng (Search per map move).

## 4. Đặc tả Kết quả Tìm kiếm (Serp View)
- **Tính năng**: 
    - Hiển thị dạng Grid/List linh hoạt.
    - Đánh dấu từ khóa khớp trong tên tour (Highlighting matches).
