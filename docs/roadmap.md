# 🗺️ Lộ trình Phát triển Dự án (Roadmap)

Kế hoạch triển khai hệ sinh thái Vivu Travel qua 6 Sprint chiến lược.

## 📅 Sprint 1: Nền tảng & Hạ tầng (In Progress)
- Thiết lập Mono-repo (Web-App-Doc).
- Cấu trúc Database & Auth (Prisma/Supabase).
- Xây dựng Documentation Portal.

## 📅 Sprint 2: Quản trị Nội dung & Điểm đến
- CRUD Điểm đến (Admin).
- Tích hợp Cloudinary Media.
- API v1 sync dữ liệu gốc.

## 📅 Sprint 3: Khám phá & Tìm kiếm
- Chức năng Discovery (Client/App).
- Search Engine (Elastic/Basic Search).

## 📅 Sprint 4: Đặt chỗ & Thanh toán
- Tour Booking Flow.
- Payment Gateway (VNPay/MoMo).

## 📅 Sprint 5 & 6: Quy hoạch & Tối ưu
- Loyalty Program.
- Performance Tuning & Release.
