# Design Inputs: Trải nghiệm Người dùng (User Experience)

## 1. User Dashboard Layout
- **Sidebar:** Hồ sơ, Chuyến đi, Yêu thích, Thông báo.
- **Booking Card:** Status badge, Ảnh tour, Ngày đi, Chi tiết tổng tiền.

## 2. Review Component
- Nhập text area + 5 slot upload ảnh.
- Star picker (1-5).

## 3. Interactions
- **Empty State:** Hình ảnh minh họa khi chưa có đơn hàng nào.
- **Rank Progress:** Thanh tiến trình hiển thị mức độ chi tiêu để lên hạng tiếp theo.
- **Animations:** Slide-in bảng điều khiển trên mobile.
