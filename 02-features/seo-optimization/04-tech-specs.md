# Tech Specs: SEO & Optimization

## 1. Next.js Features
- Sử dụng `Metadata API` của Next.js 15.
- Cấu hình `next/image` với `formats: ['image/webp', 'image/avif']`.
- Triển khai `generateStaticParams` cho các trang tour phổ biến để pre-render.

## 2. Tools
- Sử dụng `next-sitemap` để tự động tạo sitemaps.
- Kiểm tra lỗi SEO với `Ahrefs` hoặc `Screaming Frog`.
